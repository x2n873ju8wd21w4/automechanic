# AutoMech mobile (Flutter, MVP-скелет)

Статус: скелет написан, **не собирался** (на машине разработки нет Flutter SDK) —
первая сборка может потребовать мелких правок.

## Сборка

```bash
flutter create . --platforms=android,ios   # догенерит платформенные папки
flutter pub get
flutter run --dart-define=API_BASE=http://10.0.2.2:8000
# релиз на реальный API:
flutter build apk --dart-define=API_BASE=https://api.твойдомен --dart-define=API_KEY=...
```

`10.0.2.2` — localhost хоста из Android-эмулятора. Для реального устройства в
одной сети — IP машины с бекэндом.

## Экраны

1. **Поиск**: симптом + фильтры (марка/модель/DTC), список кейсов со скором,
   кнопка «Спросить ИИ» (RAG-ответ бекэнда со ссылками на кейсы).
2. **Кейс**: авто/система, проблема, причина, шаги диагностики и ремонта,
   замеры (норма/факт), нюансы; кнопки ▶ открывают YouTube прямо на таймкоде
   нужного момента.
